# Kitty Keymaps

## Intro

`KittyMod` means `Alt+Shift`.

Key `A` means to press the key `a`, not `Shift+a`.

## Clipboard

- `Ctrl + C`: Copy to clipboard when selecting or interrupt when not selecting.
- `Ctrl + V`: Paste from clipboard.

## Scrolling

`KittyMod + b`: Show scrollback

## Window management

The term "window" here doesn't means OS window, but a pane. When we infer
`OS window`, it means the real OS window.

- `KittyMod + W`: Open a new window, it will split current window into two.
- `KittyMod + N`: Open a new OS window.
- `KittyMod + D`: Close a window.
- `KittyMod + ]`: Next window.
- `KittyMod + [`: Previous window.
- `KittyMod + l`: Move window forward.
- `KittyMod + h`: Move window backward.
- `KittyMod + r`: Resize window.

## Tab management

- `KittyMod + t`: Open new tab
- `KittyMod + q`: Close tab
- `KittyMod + Right`: Next Tab
- `KittyMod + Left`: Previous Tab
- `KittyMod + .`: Move tab forward
- `KittyMod + ,`: Move tab backward
- `KittyMod + Ctrl + t`: Set tab title

## Layout management

- `KittyMod + f`: Next Layout

## Fonts

- `KittyMod + =`: Increase font size
- `KittyMod + -`: Decrease font size
- `KittyMod + Backspace`: Reset font size

## Selection and action

- `KittyMod + e`: Open url with selection

## Miscellaneous

`KittyMod + F1`: Show kitty doc
`KittyMod + F11`: Toggle fullscreen
`KittyMod + F10`: Toggle maximized
`KittyMod + F2`: Edit config file
`KittyMod + Delete`: Reset the terminal
`KittyMod + F5`: Reload kitty config
`KittyMod + F6`: Debug kitty config
