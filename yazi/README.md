# Yazi Config

This is the config for [`yazi`](https://github.com/sxyazi/yazi)

Usually, this folder should be placed at `${XDG_CONFIG_HOME}`.

## Usage

The `flavors` and `plugins` are not included here. To manage them, you can use the offical
package manager.

In this configuration, the following flavors and plugins are installed.
To quick install all of them, run [`setup.sh`](./setup.sh) in this directory.

- [Dracula Flavor for Yazi](https://github.com/yazi-rs/flavors/tree/main/dracula.yazi), licensed under MIT license.
- [full-border.yazi](https://github.com/yazi-rs/plugins/tree/main/full-border.yazi), licensed under MIT license.
- [whoosh.yazi](https://gitlab.com/WhoSowSee/whoosh.yazi), licensed under MIT license.
